# Repeated word
Write a function that takes a string as a parameter. Return the first repeated word in a string.

## Example 1:
### Input:
"ASAC is a department at LTUC. ASAC teaches programming in LTUC."

### Output:
```ASAC```

## Example 2:
### Input:
"I am learning programming at ASAC"

### Output:
```No Repetition```
